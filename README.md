# Convert XLS / XLSX To CSV

 * This program will convert XLS or XLSX to CSV file and store the CSV in the folder where the source file is present.

 ## Requirements
1. Python >= 3

## Usage

```bash
$ python3 xls_xlsx_csv.py [PATH TO THE XLS/XLSX File]
```
